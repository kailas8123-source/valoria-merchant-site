# Valoria Supply

Original tactical FPS-inspired ecommerce storefront for gaming merchandise dropshipping. The visual identity is dark, cinematic, neon-accented, and esports-oriented while avoiding official Riot Games or Valorant assets, logos, characters, maps, or copyrighted visuals.

## Features

- React, TypeScript, Vite ecommerce app
- Home, catalog, product detail, cart, checkout, and order-success routes
- Expanded product catalog for apparel, desk setup gear, wall art, stickers, and streamer bundles
- Search, category filter, bundle filter, quick view, wishlist, animated add-to-cart, and sliding cart drawer
- Mobile-first responsive styling, sticky navigation, floating CTA, and reduced-motion support
- SEO title, description, theme color, and Open Graph metadata

## Run Locally

```bash
npm install
npm run dev
```

Open `http://127.0.0.1:5173/`.

## Build

```bash
npm run build
```

## Notes

Product images are remote editorial placeholders. For production dropshipping, replace them with licensed product photography or generated original product renders before launch.
